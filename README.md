## FAQ

**How do I update the app?**  
Modify the HTML and any related front-end files, then deploy the static site again.

**Is a server-side app necessary?**  
No. The project is intended to function as a browser-based static web app.

**Where can I host it?**  
Any regular static hosting provider should work, including basic web hosting services.

**What should I check if the page fails to load?**  
Verify that the main HTML file is in the right place and that every linked asset was uploaded with it.

**Can I change the interface?**  
Yes. Since the layout is front-end based, you can edit the HTML and static assets directly.
